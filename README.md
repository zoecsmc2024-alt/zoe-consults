# 🏦 ZoeLend IQ - Zoe Consults

A professional loan management dashboard built with Python and Streamlit.

### ✨ Key Features
* **🔐 Secure Login:** Password-protected access for administrative security.
* **📈 Auto-Compounding:** Interest is calculated automatically using the formula $A = P(1 + r/n)^{nt}$.
* **🕵️ Search & Filter:** Instantly find customers by name.
* **⚠️ Dormancy Alerts:** Automatic status updates for loans with no activity for 60+ days.
* **📄 Document Generation:** One-click welcome letters for new clients.

### 🛠️ Technology Stack
* **Language:** Python
* **Deployment:** Streamlit Cloud
* **Data Storage:** CSV (GitHub-hosted)
* **Libraries:** Pandas, Streamlit, Datetime
